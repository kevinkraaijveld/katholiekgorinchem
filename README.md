# Parochie Website (MkDocs)

This project contains the source files for the parish website, built with **MkDocs** and written primarily in **Markdown**.  
The site includes parish news, articles, sacramental information, prayer resources, and practical information for parish life.

## ✝️ Purpose

The goal of this website is to:
- Share parish news and articles
- Provide clear information about the sacraments
- Support prayer and devotion
- Strengthen parish community and communication

Everything is structured to be **simple to maintain**, **easy to extend**, and **readable for non-technical editors**.
